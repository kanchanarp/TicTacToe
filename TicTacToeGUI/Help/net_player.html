• Start Network game:
    • Start two seperate instances of Tic Tac Toe
    • To play as Server
      • Go to Network -> Connect Server -> Start new server
      • To play X: Select Play X
      • To play O: Select Play O
    • To play as Client
      • Go to Network -> Connect Server -> Connect to a server
      • you will either be assigned X or O depending on the server choice
    • To disconnect connection
      • Go to Network -> Disconnect
